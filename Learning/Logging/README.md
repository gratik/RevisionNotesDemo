# Logging & Observability

Comprehensive logging patterns for ASP.NET Core applications.

## Files

- **ILoggerDeepDive.cs** - ILogger interface, scopes, log levels
- **StructuredLogging.cs** - Structured logging with Serilog
- **LoggingBestPractices.cs** - Correlation IDs, performance logging

## Key Concepts

- Structured logging vs text logging
- Log levels (Trace, Debug, Info, Warning, Error, Critical)
- Log scopes for context
- Serilog sinks and enrichers
- Application Insights integration
