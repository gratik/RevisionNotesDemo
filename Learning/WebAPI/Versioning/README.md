# API Versioning

Strategies for evolving APIs without breaking existing clients.

## Files

- **APIVersioningStrategies.cs** - URL, header, query string versioning
- **VersioningBestPractices.cs** - Deprecation, sunset headers, migration

## Key Concepts

- URL versioning (api/v1/products)
- Header versioning (API-Version: 1.0)
- Query parameter versioning (?api-version=1.0)
- Media type versioning (Accept: application/vnd.myapp.v1+json)
- Deprecation policies
